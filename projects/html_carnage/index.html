Reasons for cultivating your HTML skills

Such question is to serve as a reminder for myself, so that I would not rush in learning while not having my founcation established solidily. To answer this question, we need to know what is considered good in HTML and among all, writing semantic HTML is very important.

There are several reasons as below:
  • It makes your code easier to understand.
  • It makes the source code more readable for others.
  • Browsers can interpret your code better and therefore boosts the SEO (Search Engine Optimization), resulted in higher accessibility.

Tricks/Good Practices when writing HTML

  1. Avoid abusing <div> and <span>. There are tons of alternative with built-in features provided by HTML5 (we will cover that more in other parts on this page).
  2. Be discreet when giving name to the class/ID for elements, be specific. (this is difficult for me.. i am never good in naming).
  3. Use lowercase for elements, attributes and also values, for better readability and consistency.
  4. Use labels for bars/options in the form.
  5. Add alt (Alternative Text Attribute) on images/links as fallback.
  6. Use <h1> only once per page.

Semantic Tags

When creating a brief layout, the following tags can be used:

<header> <footer> <article> <nav> <aside> <section> <details> <summary>

Then, there are some other sematic tags for various types of text/media(image/audio/video) elements:

<em> <strong> <s> <q> <blockquote> <figure> <figcaption> <audio> <video> <source> <track>

Other Useful Tags

<time> <map>

For using a single image but mapping to different links. can be useful.

<img src="./asakusa_done.svg" alt="Asakusa Shrine - Kaminarimon (Thunder Gate)" usemap="#kaminarimon" width="400" height="379"> <map name="workmap"> <area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm"> <area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm"> </map>

First add attribute "usemap" and assign an unique ID to the image. then, using <map name={usemap}> to wrap all <area>. Followed with the above, inside <area> set the attributes, the basic ones will be: shape, coords, href and alt.
